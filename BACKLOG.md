## Tennis Prototype (`tennis.py`)

Аркадный теннис в стиле Wii Sports — управление взмахом руки через камеру.

### Core mechanics

- [ ] Мяч летит по параболе, отскакивает от стен и сетки
- [ ] Противник (AI) на правой стороне корта — простой трекинг мяча
- [ ] Удар регистрируется как быстрый горизонтальный/диагональный свайп открытой ладонью
- [ ] Направление удара = угол движения руки в момент свайпа
- [ ] Скорость мяча после удара зависит от скорости свайпа

### Удары

| Жест | Удар | Эффект |
|---|---|---|
| Свайп вправо (низко) | Форхенд | Стандартный удар |
| Свайп влево (низко) | Бэкхенд | Стандартный удар |
| Свайп вверх | Подача / смэш | Быстрый мяч сверху |
| Свайп вниз | Укороченный | Медленно, близко к сетке |

### Визуал

- [ ] Вид сбоку (2D): корт, сетка, две стороны
- [ ] Траектория мяча с лёгким arc
- [ ] Тень мяча на корте (помогает понять высоту)
- [ ] PiP камера в углу
- [ ] Счёт: геймовая система (15-30-40-game) или упрощённая (очки до 7)

### AI противник

- [ ] Простой: ракетка следует за мячом по Y с задержкой
- [ ] Иногда промахивается при высокой скорости

### Stretch goals

- [ ] Два режима: тренировка (стенка) и vs AI
- [ ] Спецудары: топспин (круговой жест) — мяч низко и быстро
- [ ] Счёт в стиле настоящего тенниса (games, sets)

## Mage Duel (`mage_duel.py`)

Два игрока на одной камере — каждый занимает свою половину экрана, кастуют заклинания друг в друга.

### Core mechanics

- [ ] Экран разделён вертикально: левый игрок vs правый игрок
- [ ] У каждого HP-бар, цель — довести противника до 0
- [ ] Каждый игрок отслеживается отдельно (MediaPipe `num_hands=2`)
- [ ] Заклинания летят горизонтально от кастующего к противнику
- [ ] Можно отбить летящее заклинание встречным (столкновение снарядов)

### Заклинания (те же жесты, что в spell_game)

| Жест | Заклинание | Эффект |
|---|---|---|
| Кулак + свайп | Огненный шар | Прямой урон |
| Открытая ладонь + вниз | Щит | Блокирует следующий снаряд |
| Палец + круг | Орб | Медленный, большой урон |
| Пинч + бросок | Стрела | Быстрая, сложно отбить |

### Визуал

- [ ] Арена с двумя магами по краям, снаряды летят между ними
- [ ] Щит отображается как полупрозрачный барьер перед игроком
- [ ] Столкновение снарядов — взрыв в центре экрана
- [ ] PiP не нужен — оба игрока видны на основном фоне камеры

### Stretch goals

- [ ] Комбо: если оба кастуют одновременно — столкновение и "pushing match" (кто сильнее давит жестами)
- [ ] Режим 1 игрок vs AI-маг
